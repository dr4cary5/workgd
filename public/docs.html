Documentation

Technical documentation for the ShadowX development environment.

📦 Installation

git clone https://github.com/dr4cary5/shadowx.git
cd shadowx
npm install

🚀 Deployment

This project is configured for automatic deployment on Vercel. Push to the main branch to trigger a production deployment.

git push origin main

⚙️ Configuration

Environment variables are managed through Vercel's dashboard. See vercel.json for build and routing configuration.

{
  "functions": {
    "api/*.go": {
      "memory": 512,
      "maxDuration": 10
    }
  }
}

🧪 Testing

vercel dev        # Local development
vercel deploy     # Preview deployment
vercel --prod     # Production deployment
